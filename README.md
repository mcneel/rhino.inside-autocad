# Rhino.Inside® Autocad


Please see the [Rhino.Inside.Autocad Forum](https://discourse.mcneel.com/t/about-the-autocad-category/213088)
to find out more about the project and find out how to get involved.

## Overview

The Rhino.Inside® Technology allows Rhino, Grasshopper, and their add-ons to be
embedded within other products.
This repository is the Rhino.Inside® for Autodesk AutoCAD, named Rhino.Inside.AutoCAD.

## Alpha Testing

This project is in early Alpha testing, and it is not recommended for use in a production environment.
